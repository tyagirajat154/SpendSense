# SpendSense: Data-Driven Personal Expense Analysis

## Overview
SpendSense is a Data Science project that analyzes personal spending patterns
using data visualization and statistical analysis. The goal is to identify
spending trends, category-wise expenses, and monthly financial behavior.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains the following fields:
- Date
- Category
- Amount
- Payment Method
- Description

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights Generation


## Key Analysis
- Category-wise expense distribution
- Monthly spending trends
- Payment method analysis

## Visualizations
- Pie Chart (Expense distribution)
- Bar Chart (Category spending)
- Line Graph (Monthly trends)

## Conclusion
The SpendSense analysis reveals that most expenses occur in categories
such as groceries, shopping, and transportation. Visualization of monthly
spending trends helps identify high-expense periods and enables better
financial planning.
